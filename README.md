# Creating a new blog post

Use the following template for starting the post
```
---
title: "Your Post Title"
description: "Brief description"
pubDate: 2024-12-30
author: "Your Name"
tags: ["tag1", "tag2"]
---

# Your Title

Your content here...
```

Once done commit you changes and that is all that is needed.
